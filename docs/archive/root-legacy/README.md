# Root Legacy Archive

Thư mục này chứa các file từng nằm ở root repo nhưng hiện đã được archive để làm gọn cấu trúc và giảm nhầm lẫn giữa:
- tài liệu active đang điều hành repo,
- tài liệu historical/snapshot,
- prompt/plan cũ chỉ còn giá trị tham khảo.

## Ví dụ nội dung trong đây
- master plan cũ
- changelog/progress cũ
- handover snapshot cũ
- install guide cũ
- prompt điều phối batch cũ
- audit/backlog lịch sử không còn là source of truth active

## Quy tắc
- Nếu cần hiểu lịch sử, có thể đọc các file trong đây.
- Nếu cần biết trạng thái hiện tại của repo, quay lại `CANONICAL_DOCS.md` ở root.
